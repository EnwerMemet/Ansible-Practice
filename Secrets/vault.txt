Ansible Vault

Ansible Vault encrypts variables and files so you can protect sensitive content such as passwords, API keys, or secret tokens.


1. Create an Encrypted File
To create a brand new secret file, use the create command.

Bash
ansible-vault create secret_vars.yml
What happens next:

Ansible will ask you for a New Vault password. Pick something you'll remember!

It will open your default terminal editor (usually vi or nano).

Type your secrets in YAML format, for example:

YAML
db_password: "SuperSecretPassword123"
api_key: "ABC-123-XYZ"
Save and exit. If you look at the file now (cat secret_vars.yml), it will look like a bunch of gibberish text.

1. How to Save and Exit Vim
Press the Esc key (to make sure you aren't in "Insert" or "Recording" mode).

Type exactly this: :wq

Press Enter.
